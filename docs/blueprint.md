# **App Name**: Personal Canvas

## Core Features:

- Profile Showcase: Display personal information, including a profile picture, bio, and contact details.
- Blog Posts Display: Display a collection of blog posts with titles, summaries, and links to full articles.
- Project Gallery: Display links to projects and work samples, with descriptions and visuals.
- Backend Management: Ability to add new projects and blogs from a back end where i need not write code everytime.

## Style Guidelines:

- Primary color: Neutral beige or light gray for a clean and modern look.
- Secondary color: Soft blue or green for a touch of personality and calmness.
- Accent: Coral or teal to highlight important sections and calls to action.
- Clean and modern sans-serif fonts for headings and body text.
- Simple and elegant line icons for navigation and visual aids.
- Clean and modern grid-based layout with clear sections.
- Subtle transitions and animations for a smooth user experience.
